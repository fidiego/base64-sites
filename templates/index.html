Base64 Sites

Have you ever wanted to encode a tiny website in base64 but needed the link to be shared in mobile or otherwise handled by devices that only know about well-formed URLs? Well, me too.

This site is my answer to that problem.

How to use

How it works

Inspect this document and check out the contents of onSubmit or copy and paste the following in the developer console: 

document.getElementById('logic').text

  1. Base64 encode the HTML string and add the data:text/html;base64, prefix.
  2. URI Encode the base64 string with encodeURIComponent
  3. Pass URI encoded string to /render as the context query param.
  4. Profit?
